# BuffCheckByFerocious

BuffCheckByFerocious is a lightweight World of Warcraft addon designed for raid and party leaders to monitor essential buffs and consumables in real-time.

## Features

- Smart Scan: Monitors Class Buffs, Flasks (Standard and Midnight expansion), and Augment Runes.

- Quality Detection: Automatically differentiates between high-quality (Green) and lower-quality (Yellow) consumables.

- Dynamic UI: A clean, auto-resizing window that hides during combat to save resources.

- Reporting: Quickly report missing buffs to Raid or Party chat with a single click.

- Optimized for Midnight: Specifically updated for World of Warcraft Aura IDs.

## How to use

- Lock/Unlock: Click the lock icon to prevent moving the frame.

- Transparency: Use the mouse wheel over the title to adjust background alpha.

- Reporting: Click the scroll icon to list players missing buffs in chat.

- Debug: Type /buffcheck debug to see active Aura IDs on your character.

# Installation

Extract the BuffCheckByFerocious folder into your `World of Warcraft/_retail_/Interface/AddOns/BuffCheckByFerocious` directory.
